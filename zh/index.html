工作原理

只需三步，从游戏到病毒式内容

1

玩 CS2

在比赛前启动 TickZero 的实时记录模式。 系统在第一回合开始时自动开始录制，并以精确的时间戳捕捉每一次击杀、爆头和残局时刻。

python launcher.py --record
2

AI 分析

比赛结束后，Google Gemini AI 会分析您的游戏日志以识别值得记录的高光时刻。 它会优先考虑五杀、多杀、残局、连杀爆头和高技术操作。

  • 🔥 多杀 (2K, 3K, 4K, ACE)
  • 💪 残局 (1v2, 1v3, 1v4)
  • 🎯 连续爆头
  • ⚡ 极限残血反杀
3

导出与分享

TickZero 自动生成针对 TikTok 和 Reels 优化的竖屏 (9:16) 片段。 每个剪辑都经过硬件加速和专业格式化，准备好可以直接上传。

分辨率: 1080×1920
时长: 8-15 seconds
格式: MP4 (H.264)

关于 TickZero 你需要知道的一切

是的！TickZero 是 100% 免费并在 MIT 许可下开源的。它使用 Google Gemini 的免费层级（每天 1500 次请求），足以满足每天约 50 场比赛的需求。无需信用卡，无隐藏费用。

你需要：

  • Python 3.10 或更高版本
  • 安装了 WebSocket 插件的 OBS Studio
  • Counter-Strike 2
  • FFmpeg (用于视频处理)
  • 免费的 Google Gemini API 密钥

所有安装说明均包含在文档中。

Google Gemini 分析你的游戏日志并根据以下内容确定优先级：

  • 多杀： 双杀、三杀、四杀和五杀
  • 残局： 人数劣势获胜
  • 爆头精度： 精准的一枪毙命
  • 高技术操作： 快速反应和高难度射击
  • 残血反杀： 极低生命值存活

你可以调整最低优先级阈值来控制 AI 的筛选严格程度。

目前，TickZero 专为通过 WebSocket 与 OBS Studio 配合使用而设计，以实现自动录制控制和精确的时间戳同步。不过，你可以手动使用任何软件录制并稍后处理视频 - 你只需要确保事件日志的时间戳与录制开始时间对齐。

是的！TickZero 是跨平台的，可以在 Windows、Linux 和 macOS 上运行。唯一的要求是你的系统上安装了 Python 3.10+、OBS Studio 和 FFmpeg。CS2 游戏状态集成也适用于所有平台。

TickZero 仅捕捉你的个人击杀。当你死亡并观看队友时，系统会利用 SteamID 追踪自动过滤掉他们的击杀。这确保了精准的个人高光追踪并防止误报。

处理时间取决于你的硬件和剪辑数量：

  • AI 分析： 每场比赛 5-10 秒
  • 视频处理 (GPU)： 每个剪辑 2-5 秒
  • 视频处理 (CPU)： 每个剪辑 10-20 秒

一场典型的有 3-5 个高光的比赛在使用 GPU 加速的情况下不到 1 分钟即可处理完毕。

目前，剪辑已针对 TikTok/Reels 进行优化（8-15 秒，1080×1920）。自定义持续时间和格式已列入 v2.0 的路线图。如果您熟悉 Python，可以修改源代码以调整这些参数。
