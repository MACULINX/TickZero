## Tips for Logo Usage:
- The logo should work well at small sizes (favicon, app icon)
- Ensure good contrast for both light and dark backgrounds
- Keep it simple enough to be recognizable
- Make sure it scales without losing detail
- Test readability at 16x16, 32x32, and larger sizes

## Logo File Formats Needed:
- PNG (transparent background) - for web use
- SVG (vector) - for scalability (can be created from PNG if needed)
- ICO - for favicon (can convert from PNG)
